# Release Notes - v0.12.0

## 🎉 新機能

### 📱 モバイル対応
- **Android版リリース**: スマートフォンとタブレットで箱書きエディタが使えるようになりました
- **レスポンシブデザイン**: 
  - スマートフォン（縦画面）: 1列の縦スタックレイアウト
  - タブレット（横画面）: PC版と同じグリッドレイアウト
- **セーフエリア対応**: ノッチやステータスバーに被らないUI設計
- **モバイル専用機能**: アプリのDocumentsフォルダへの自動書き出し

### 💾 デスクトップ版の改善
- **終了時の保存確認**: アプリを閉じる際に保存を確認するダイアログを追加
- **自動保存機能**: JSON保存とテキスト書き出しを自動実行

## 🐛 バグ修正
- ライトモード時に裏設定欄がダークモードのままになる問題を修正
- モバイルでヘッダーがシステムUIに被る問題を修正

## 📦 ダウンロード

### デスクトップ版
- **macOS**: `箱書きエディタ_0.12.0_universal.dmg`
- **Windows**: `箱書きエディタ_0.12.0_x64-setup.exe`
- **Linux**: `箱書きエディタ_0.12.0_amd64.deb`

### モバイル版
- **Android**: `app-universal-debug.apk` (野良アプリ配布用)
  - インストール方法: 端末の設定で「提供元不明のアプリ」を許可してインストール

## 🔧 技術的な変更
- Tauri Mobile対応の実装
- レスポンシブCSSの追加
- プラットフォーム自動検出機能の実装
