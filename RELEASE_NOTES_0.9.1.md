# 箱書きエディタ v0.9.1

## 新機能

### テーマ選択機能
- **システムデフォルト**、**ライトモード**、**ダークモード**の3つのテーマから選択可能
- 設定メニューから簡単に切り替えられます
- ユーザーの好みに合わせた快適な執筆環境を提供

## 改善

### シーングリッド表示の最適化
- シーンカードの表示を簡素化し、より見やすく
- **シーンタイトル**、**章名**、**あらすじの一部**のみを表示
- 不要な情報を削減し、執筆フローに集中できるように改善

### 自動リリースワークフロー
- GitHub Actionsによる自動ビルドシステムを導入
- Windows、macOS (Universal)、Linux向けのビルドを自動生成
- より迅速で安定したリリースプロセスを実現

## ダウンロード

### macOS (Universal)
- Apple Silicon (M1/M2/M3) と Intel Mac の両方に対応
- `箱書きエディタ_0.9.1_universal.dmg` をダウンロード

### Windows
- `箱書きエディタ_0.9.1_x64-setup.exe` (64bit版インストーラー)
- `箱書きエディタ_0.9.1_x64_en-US.msi` (MSIインストーラー)

### Linux
- `箱書きエディタ_0.9.1_amd64.deb` (Debian/Ubuntu)
- `箱書きエディタ_0.9.1_amd64.AppImage` (汎用)

## 技術的な変更

- TypeScriptの型エラーを修正
- 未使用パラメータの削除によるコード品質の向上
- ビルドプロセスの自動化

## インストール方法

### macOS
1. DMGファイルをダウンロード
2. DMGファイルを開く
3. アプリケーションフォルダにドラッグ&ドロップ

### Windows
1. セットアップファイル(.exe)またはMSIファイルをダウンロード
2. ダウンロードしたファイルを実行
3. インストールウィザードに従ってインストール

### Linux
**Debian/Ubuntu:**
```bash
sudo dpkg -i 箱書きエディタ_0.9.1_amd64.deb
```

**AppImage:**
```bash
chmod +x 箱書きエディタ_0.9.1_amd64.AppImage
./箱書きエディタ_0.9.1_amd64.AppImage
```
