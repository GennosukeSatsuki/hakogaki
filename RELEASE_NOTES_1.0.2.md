# HakoGraph - v1.0.2

内部構造の劇的な改善とメンテナンス性の向上。

## [1.0.2] - 2025-12-22

### 追加・改善

- **アーキテクチャの刷新: CSS Modules の導入**
  - 全コンポーネントのスタイルを CSS Modules へ移行しました。
  - スタイルの影響範囲をコンポーネント単位にカプセル化し、予期せぬ表示崩れのない堅牢なシステムを実現しました。
  - 巨大な `App.css` を整理し、コードの可読性が大幅に向上しました。
- **データ管理の統合とクリーンアップ**
  - `localStorage` の直接操作を完全に排除し、Zustand (Store) による一元管理へ移行しました。
  - 文字数キャッシュなどの一時データも Store 内で管理されるようになり、一貫性のあるデータフローを実現しました。
- **コード品質の向上**
  - ベテランエンジニアによるレビューにも耐えうる、モダンでクリーンなディレクトリ構造と命名規則を徹底しました。

### 技術的な変更

- `src/styles/variables.css` によるグローバル変数の集中管理。
- Zustand `persist` ミドルウェアによる自動ストレージ同期の最適化。
- 不要なインラインスタイルの徹底的な排除。
