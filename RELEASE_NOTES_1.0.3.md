# HakoGraph Release Notes v1.0.3

今回のアップデートでは、構成（箱書き）と本文の書き出し処理、およびエディタでの判定ロジックを大幅にリファクタリングしました。

## 🛠 改善・修正
- **書き出し処理の共通化**: プロジェクトの一括書き出しとエディタでの個別保存・判定ロジックを共通化しました。これにより、一貫した動作を保証します。
- **セパレーター判定の強化**: 本文と箱書きを分けるセパレーターの判定に正規表現を導入し、全角ダッシュやハイフンなどの文字の揺れを柔軟に許容するようになりました。
- **意図しない上書きの防止**: エディタと書き出し処理の間で「何が本文か」の定義が不一致だった問題を修正し、箱書き部分が誤って消去されるリスクを解消しました。
- **パス解決の安定化**: OSごとのファイルパスの扱いや、チャプター名・ファイル名の生成ロジックを統一し、保存の確実性を高めました。

## 📦 内部的な変更
- `src/utils/exportUtils.ts` へのロジック集約
- エディタ側のコード簡略化と文字数カウントの正確性向上
