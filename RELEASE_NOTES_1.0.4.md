# HakoGraph Release Notes v1.0.4

今回のアップデートでは、ユーザーインターフェースの柔軟性向上と、自動書き出し機能の安定性を強化しました。

## ✨ 新機能
- **シーン入力画面のフォントサイズ設定**: 設定の「全般」タブから、シーン入力画面（タイトル、あらすじ、裏設定など）のフォントサイズを自由に変更できるようになりました。執筆環境に合わせて調整可能です。

## 🛠 修正・改善
- **セパレーター重複出力の修正**: 特定の条件下で自動書き出しが行われた際に、セパレーターが二重に出力されてしまう不具合を修正しました。
- **セパレーター判定の精度向上**: 箱書き（メタデータ）と本文を分けるセパレーターの検出ロジックを強化し、内容にダッシュが含まれている場合でも正しく分割できるようになりました。
- **書き出し時のフォーマット修正**: 箱書きと本文の間の不要な空行を整理しました。
