# HakoGraph v1.0.6 リリースノート

**リリース日**: 2026年1月3日

## 🆕 新機能

### About画面にオープンソースライセンス情報を追加

「バージョン情報」（About）画面から、アプリで使用しているオープンソースライブラリのライセンス情報を確認できるようになりました。

**表示されるライブラリ:**
- React (MIT)
- Tauri (MIT/Apache-2.0)
- @dnd-kit (MIT)
- Tiptap (MIT)
- i18next (MIT)
- Zustand (MIT)
- React Router (MIT)

各ライブラリ名をクリックすると、GitHubリポジトリに移動できます。
